[Задание взято тут.](https://github.com/sergei-maksatpro/test-task-c-sharp)


Замечания:
1. Не сказано, как детектировать и обработать возможное пустое значение.
   Соответственно, такая ситуация пропущена.
2. В выражении '=A2B1' пропущен знак операции. Будем считать, что это умножение.
   Но обработается и ситуация с '=A2*B1'.
3. Вместо электронной таблицы запускается создание локальной БД и её заполнение полученными данными.
4. Заменить connectionString на свой вариант.
5. Как заводить из ком. строки

   - cd XXXXXXX\Test-task-for-CSharp-developper\ConsoleApp1\ConsoleApp1\bin\Debug\net9.0
   
   - ConsoleApp1.exe [аргументы через запятую]
